#  Notes

## UNIX commands

* "list files" = ls
* "change directory" = cd dir
* "present working directory" = pwd
* "make directory" = mkdir name
* "remove" = rm file
* "recursive forced remove" = rm -rf dir

## Git (Distributive Version Control)
* cd project
* git init (first time)
* git status
* git add . or git add file
* git commit
* git commit -m "Message"
